# Tab Manager - 功能增强更新

## 🎉 新增功能概览

本次更新参考了 Toby Chrome 扩展的核心功能与 UI 设计理念，为用户带来更现代化的标签页和书签管理体验。

---

## ✨ 核心功能

### 1. **集合/工作空间管理 (Spaces)**
- ✅ 创建多个独立的 Space 来组织不同的工作或项目
- ✅ 每个 Space 可以包含多个分组 (Groups)
- ✅ 支持删除不需要的 Space
- ✅ 快速切换不同 Space

**使用方式:**
- 点击顶部导航栏的 `+` 按钮创建新 Space
- 点击 Space 名称切换工作空间
- 鼠标悬停在 Space 上显示删除按钮 (×)

---

### 2. **卡片式布局设计**
- ✅ 采用现代化卡片式设计，视觉效果更佳
- ✅ 每个分组以独立卡片形式展示
- ✅ 卡片支持自定义背景色
- ✅ 响应式网格布局，自适应屏幕尺寸

**视觉特性:**
- 圆角卡片设计
- 优雅的阴影效果
- 悬停动画反馈
- 渐变色彩按钮

---

### 3. **深色模式切换**
- ✅ 一键切换深色/浅色模式
- ✅ 自动保存用户偏好设置
- ✅ 完整的深色主题配色方案
- ✅ 保护眼睛，适合夜间使用

**使用方式:**
- 点击右上角的 🌙/☀️ 按钮切换模式
- 系统会自动记住您的选择

---

### 4. **增强的拖拽交互体验**
- ✅ 支持拖拽书签到不同分组
- ✅ 支持拖拽调整分组顺序
- ✅ 支持从侧边栏拖拽标签页到分组
- ✅ 流畅的拖拽视觉反馈

**拖拽功能:**
- 拖拽时显示半透明效果
- 目标位置高亮提示
- 平滑的动画过渡效果
- 防止误操作的延迟机制

**操作示例:**
```
1. 拖拽左侧边栏的标签页 → 任意分组卡片
2. 拖拽分组标题 → 调整分组排序
3. 拖拽书签卡片 → 调整书签顺序
```

---

### 5. **笔记和分享功能**

#### 笔记功能
- ✅ 每个书签支持添加备注/简介
- ✅ 编辑模式下可输入详细描述
- ✅ 笔记内容自动保存在书签中
- ✅ 支持编辑和修改

**使用方式:**
- 点击书签上的 ✏️ 编辑按钮
- 在"简介"字段输入备注内容
- 点击保存即可

#### 分享功能
- ✅ 导出 Space 数据为 JSON 文件
- ✅ 从 JSON 文件导入 Space
- ✅ 生成分享链接 (模拟功能)
- ✅ 复制分享链接到剪贴板

**使用方式:**
- 点击分组上方的"🔗 分享"按钮
- 或在底部工具栏使用"导入/导出"功能

## 🔧 技术实现

### 新增组件
- `ThemeToggle.tsx` - 深色模式切换按钮
- 增强的 `BookmarkCard.tsx` - 卡片式书签组件
- 增强的 `Bookmarks.tsx` - 支持分组拖拽
- 增强的 `ImportExport.tsx` - 完整的数据导入导出

### Store 增强
- `toggleDarkMode()` - 切换深色模式
- `deleteSpace()` - 删除 Space
- `moveGroup()` - 移动分组排序
- `shareSpace()` - 分享 Space
- `importSpace()` - 导入 Space 数据

### CSS 变量
```css
/* 浅色模式 */
--bg-primary: #f8f9fa;
--text-primary: #1f2937;
--accent-color: #4f46e5;

/* 深色模式 */
.dark-mode {
  --bg-primary: #1a1a1a;
  --text-primary: #f3f4f6;
  --accent-color: #6366f1;
}
```

## 📱 使用指南

### 快速开始
1. **创建第一个 Space**
   - 点击顶部的 `+` 按钮
   - 输入 Space 名称
   - 选择图标和颜色

2. **添加分组**
   - 点击"新建分组"按钮
   - 输入分组名称
   - 使用颜色选择器设置背景色

3. **添加书签**
   - 点击分组内的"+ 添加书签"按钮
   - 或直接拖拽左侧边栏的标签页

4. **整理顺序**
   - 拖拽分组标题调整分组顺序
   - 拖拽书签调整书签顺序

5. **切换模式**
   - 点击右上角的 🌙 按钮进入深色模式
   - 再次点击 ☀️ 返回浅色模式

### 高级功能
- **批量保存**: 输入会话名称，点击"保存会话"保存当前所有标签
- **数据备份**: 点击底部"导出数据"保存所有 Spaces 到 JSON 文件
- **恢复数据**: 点击"导入数据"选择之前导出的 JSON 文件
- **分享协作**: 点击"🔗 分享"生成分享链接
